What is InnoDB

主要特点：

  1. 必须要有主键，有了主键之后检索速度会得到很大提升
  2. 支持外键（MyISAM就不支持）
  3. 5.5.5之后成为MySQL的默认引擎
  4. 锁行。（MyISAM锁表）
  5. 支持事务。（MyISAM不支持）默认来说每条sql语句都会被 自动包装 成一条事务。所以多条SQL语句最好放在begin和commit之间，一起提交。
